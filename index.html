Podemos usar git para trabajar con compañeros y guardar copias de seguridad de cada uno de los cambios que vamos agregando en el código. Trabajamos en remoto.

Ventajas de usar Git

    • Combinar el trabajo de un equipo
      Trabajar en remoto
      Crear copias de seguridad con cada cambio en el código

    • Principales comandos git

    Comando Explicacion
    git add Añadir cambios
    git push Añadir los cambios a la raiz principal
    git commit Añadir comentarios
    git clone clonar en local
    git checkout master volver a rama master
    git pull descargar en visual rama master
